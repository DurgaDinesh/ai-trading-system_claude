# AI Intraday Trading System

## Project Goal

Build an AI-powered intraday trading system that analyzes:

* Market Structure
* Price Action
* Volume
* VWAP
* Support & Resistance
* News Sentiment
* Economic Events

and generates high-probability trading opportunities while maintaining strict risk management.

---

## Objectives

1. Identify high-probability intraday trades.
2. Filter low-quality setups.
3. Maintain consistent risk management.
4. Use AI to analyze trade outcomes.
5. Improve strategy performance through paper trading.
